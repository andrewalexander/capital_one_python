Deposit¶

class nessie.deposit.Deposit¶

Deposit class - get/create/update deposits

create_deposit(acc_id, deposit)¶

Create a new deposit to an acc_id

Format of POST request: ‘deposit’: {

‘medium’: ‘balance’ | ‘rewards’, ‘transaction_date’: ‘YYYY-MM-DD’, ‘status’: ‘pending’ | ‘cancelled’ | ‘recurring’, ‘amount’: int, ‘description’: ‘string’

}

Parameters:
  • acc_id – ID of the account to dump the new deposit into
  • deposit – dict containing the new deposit metadata
Returns:

dict of status code (201) and response object from Nessie
delete_deposit(deposit_id)¶

Delete a deposit

Parameters:deposit_id – ID of the deposit to delete
Returns:dict of status code (204)
get_all_by_account_id(acc_id)¶

Get all deposits for given account ID

Parameters:acc_id – ID of account to fetch deposits for
Returns:dict of status code (200) and list of deposits
get_one(deposit_id)¶

Get deposit description for given deposit ID

Parameters:deposit_id – ID of the deposit to fetch
Returns:dict of status code (200) and deposit description
update_deposit(deposit_id, deposit)¶

Update metadata associated with a deposit

Note

This will return a 404 if the deposit status is ‘executed’ or ‘cancelled’, so it should only be called immediately after a ‘create’.

Format of PUT request: ‘deposit’: {

‘medium’: ‘balance’ | ‘rewards’, ‘amount’: int, ‘description’: ‘string’ }

}

Parameters:
  • deposit_id – ID of the deposit to update
  • deposit – dict containing the updated deposit metadata
Returns:

dict of status code (202) and response message from Nessie
